
Calcit Workflow
----

> running in both Calcit and Calcit-js, with hot code reload.

### Usages

Install [Calcit](https://github.com/calcit-lang/calcit) to run demo:

```bash
cr -1 # run once

cr # run and watch
```

run tests:

```bash
cr -1 --entry test
```

run test in JavaScript:

```bash
cr --emit-js -1 --entry test # emit-js once
node main.mjs # run code
```
