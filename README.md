
Arithmetic
----

> for complex numbers

### Usages

```cirru
ns demo :require
  arithmetic.complex :as complex

complex/&+ ([] 1 2) ([] 3 4)
```

Functions:

- `&+ a b`
- `&- a b`
- `&* a b`
- `&/ a b`
- `conjugate a`
- `scale a x-scalar`
- `divide-by a x-scalar`, scales down by factor
- `polar-point radian radius`, create a point from radian
